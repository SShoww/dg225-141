## Agent skills

### Issue tracker

Issues and PRDs are tracked locally as markdown files under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage is tracked via `Status:` lines in issues matching five canonical roles (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Domain layout is single-context (one CONTEXT.md and docs/adr/ directory at the repo root). See `docs/agents/domain.md`.
